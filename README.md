# .dotfiles

Includes settings for:

| Name     | Description          |
| :------- | :------------------- |
| bash     | Shell                |
| git      | Content tracker      |
| groff    | Typesetting system   |
| irssi    | IRC client           |
| mongodb  | NoSQL Database       |
| mpv      | Media player         |
| mutt     | Email client         |
| mvi      | Image viewer         |
| newsboat | RSS client           |
| nvim     | Text editor          |
| ranger   | Filemanager          |
| rtorrent | Torrent              |
| tmux     | Terminal multiplexer |
| vim      | Text editor          |
| zathura  | PDF reader           |

## Requirements

- [GNU Make](https://www.gnu.org/software/make/)
- [GNU Stow](https://www.gnu.org/software/stow/)

## Installation

```bash
make
```
