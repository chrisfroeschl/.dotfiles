# .dotfiles

Includes settings for:

| Name        | Description          |
| :---------- | :------------------- |
| bash        | Shell                |
| git         | Content tracker      |
| groff       | Typesetting system   |
| irssi       | IRC client           |
| mongodb     | NoSQL database       |
| mpv         | Media player         |
| mutt        | Email client         |
| nnn         | File manager         |
| qutebrowser | Web browser          |
| rtorrent    | Torrent              |
| sfeed       | RSS parser           |
| tmux        | Terminal multiplexer |
| vim         | Text editor          |
| zathura     | PDF reader           |

## Requirements

- [Make](https://www.gnu.org/software/make/)
- [Stow](https://www.gnu.org/software/stow/)

## Installation

```sh
make
```
