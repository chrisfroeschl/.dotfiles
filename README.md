# dotfiles

## Description

A collection of several dotfiles for daily used CLIs.

Includes settings for:

- bash (shell)
- mongo (database)
- rtorrent (torrent)
- groff (markdown)
- htop (system)
- irssi (IRC client)
- mpv (video/audio player)
- neofetch (system)
- newsboat (RSS client)
- tmux (terminal multiplexer)
- zathura (PDF reader)
